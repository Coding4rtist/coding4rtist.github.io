***A simple but frenzied platform infinite runner.***

In caverunner you have to run out of a mysterious cave avoiding traps and at the same time escaping from mosters.

#Controls
- ◉ Tap on the right side of the screen -> JUMP
- ◉ Tap on the left side of the screen -> SHOOT

# Features
- ◉ Each time you restart, you'll play a new, randomly-generated game
- ◉ Create and edit color palettes
- ◉ Color Editor to find and edit colors without using other softwares
- ◉ Shortcuts to help you pick colors faster
- ◉ Choose from many formats (HEX,RGB,HSL,HSV)
